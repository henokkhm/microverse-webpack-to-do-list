# Microverse Webpack Template <a name="about-project"></a>

This project contains a basic Webpack template which can be used as a starting point for Microverse projects. It also sets up linters for HTML, CSS and JavaScript.

### Key Features <a name="key-features"></a>

- **Webpack:** is used to bundle JavaScript files.
- **HtmlWebpackPlugin:** is used to automatically generate the index.html file in the /dist directory.
- **webpack-dev-server :** improves developer experience by watching your source files, automatically compiling distribution files and refreshing the browser every time you *save* changes in the source code.

## ğŸ’» Getting Started <a name="getting-started"></a>

To use this template, you should first fork this repository. After that, whenever you want to use this boilerplate, you can select it as a template when you create a new repository on GitHub.

### Prerequisites

In order to use this boiler plate you need:

- <a href="https://nodejs.org/en/download">Node.js</a>
- <a href="https://docs.npmjs.com/downloading-and-installing-node-js-and-npm">npm</a>

### Setup

Clone this repository to your desired folder:

```sh
  cd desired-folder
  git clone url-to-your-repository-that-was-created-using-this-template
```

### Install

Install the dependencies with:

```sh
  cd your-local-copy-of-your-repository
  npm install
```

### Usage

To run the project, run `npm run dev`

To run the linters, run `npm run lint`

To generate static assets using Webpack, run `npm run build`


### Run tests

There are currently no tests in this project.

### Deployment

You can deploy the static assets generated by Webpack using any web-hosting service you choose.
